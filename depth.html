Set & Update Data


  1. Set - will completely overwrite all of the data stored at the node.
  2. Update - will only replace the fields that have changed at the node, or add new fields if they aren't present
  3. If you're collecting data from a sensor, you probably want to use update.



Get a Datum (single data point)

  1. Retrieve an entire month's set of data and display in a table

Day Temperature (°F)



Delete Data


  1. This function will allow you to delete a single day's data
  2. To delete an entire month's data, you need to remove modify the directory path removing the 'day' part.
